How to use the extension

Welcome to the support page for the AI‑Text‑Expander extension. This tool makes it easy to expand text using configurable shortcuts directly in the Chrome browser.

To get started, install the extension and set your shortcuts in the main interface. If you have questions about creating or editing shortcuts, check the frequently asked questions section below.

Frequently Asked Questions (FAQ)

How do I add a new shortcut?

Open the extension, click “Add new”, enter the shortcut and the expanded text, and click save.

Why doesn’t the expansion work in some fields?

Some websites use custom fields that are not properly recognized — we are working to improve this.

Can I sync shortcuts across devices?

Yes — the extension uses Chrome Sync to store data in the Chrome cloud when the user is signed into the browser.
